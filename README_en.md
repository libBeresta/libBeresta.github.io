# libBeresta

[libBeresta](https://github.com/libBeresta/libBeresta/) is a free, open-source, cross-platform library written in ANSI C for generating PDF files.

![Beresta](beresta.png)

The library allows you to create PDF documents without the need for an in-depth understanding of the internal structure of the Portable Document Format. 
It supports generating PDFs with lines, text, and images, adding various types of annotations, document compression, and embedding images in PNG, JPEG, and TIFF (Raw) formats.
It is also possible to create encrypted files and use various encodings, including UTF-8 and Asian languages (Traditional and Simplified Chinese, Korean, Japanese).

This is a modern, actively developed library with a clear roadmap that includes support for PDF 1.7 and PDF 2.0 standards.
The library is built on the principles of modularity and a clear separation between public and private APIs. Full documentation and testing of the library are planned.

The project is released under the open MIT License, making it available for commercial use.

The project is intended for developers who need a lightweight, cross-platform library for PDF generation in C. Its modular architecture allows you to use only the necessary components, making the library flexible and efficient for various use cases.

An important feature of the library is a set of data files for generating language bindings, eliminating the labor-intensive process of manually creating bindings for languages other than C.
